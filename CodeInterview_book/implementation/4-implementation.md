# Implementation (구현)

- 알고리즘은 간단한데 코드가 지나칠 만큼 길어지는 문제
- 특정 소수점 자리까지 출력해야 하는 문제
- 문자열이 입력인데, 한 문자 단위로 끊어서 리스트에 넣어야하는(파싱해야하는)문제.
<br/>
<br/>
- 파이썬 표준 라이브러리를 사용할 수 있는것들은 전부 사용하는게 좋다.
- 완전탐색, 시뮬레이션 유형.
<br/>
<br/>

### Python 에서 리스트 크기
